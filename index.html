Frequently asked

Where are my API keys stored?
Keys and OAuth tokens live in the iOS Keychain on your device. Model Pulse has no server and collects no data.

Which key does OpenAI need?
Organization spend and rate-limit data require an OpenAI admin key (starts with sk-admin-). Standard keys cannot read organization usage endpoints.

The app shows sample data — why?
Demo mode is on by default for new installs so you can explore first. Turn it off in Settings → Demo mode once your own keys are connected.
